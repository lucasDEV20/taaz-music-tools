# 🎹 Taaz Music Tools 🎧

Este repositório reúne ferramentas desenvolvidas em **Python** para otimizar o fluxo de trabalho de produtores musicais, com foco especial na cena **Psytrance** e usuários de **Ableton Live**.

O objetivo é unir a lógica de programação com a criatividade do estúdio, automatizando tarefas repetitivas.

---

## 🚀 Ferramentas Disponíveis

### 1. 🥁 Sample Organizer (`sample_organizer.py`)
Cansado de pastas de samples bagunçadas? Este script organiza seus arquivos automaticamente.
* **Como funciona:** Ele lê os nomes dos arquivos e, ao identificar palavras como "kick", "bass", "fx" ou "synth", move o arquivo para uma pasta categorizada.
* **Vantagem:** Organiza centenas de samples em segundos, deixando sua biblioteca pronta para o workflow no Ableton.

### 2. ⏱️ BPM Tapper (`bpm_analyzer.py`)
Um utilitário rápido para descobrir o BPM de qualquer som.
* **Como funciona:** Você pressiona a tecla `ENTER` no ritmo da música e o script calcula a média de BPM em tempo real.
* **Vantagem:** Leve, rápido e roda direto no terminal.
