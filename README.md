# 🎹 Taaz Music Tools

Este repositório contém scripts em Python desenvolvidos para otimizar o workflow de produção musical, focado em Psytrance e integração com o Ableton Live.

## 🚀 Ferramentas Disponíveis

### 1. BPM Tapper (`scripts/bpm_analyzer.py`)
Um utilitário de terminal para calcular o BPM de uma track ou loop em tempo real. Útil para referências rápidas durante a criação de grooves.
